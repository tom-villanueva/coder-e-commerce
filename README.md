# coder-e-commerce

The project was bootstrapped with

- Vitejs

And it's made with

- React.js
- Tailwindcss

## Extra libraries

- SweetAlert2 for confirmation and info. display alerts. To get better UX.

## Instructions

### To install the dependencies run

```bash
npm install
```

### To start the server run

```bash
npm run dev
```

### The server starts in

http://localhost:5173
